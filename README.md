# Overview

MineJack is a game that draws inspiration from popular online casino slot games such as Mines, Limbo, Crash, Plinko and Dice.

# Goal

The goal of the game is to climb up the tower by finding the hidden keys while avoiding the bombs.

# What I like most about the code

I appreciate how clean the architecture and code are, which makes them easier to understand and work with. The structure is clear, and there is good separation of concerns.

# What I would improve

While the code is well organised and easily extendable, certain critical functions could be optimised for better performance. Since I'm working with a grid, I often find myself using nested loops, which can be inefficient.
